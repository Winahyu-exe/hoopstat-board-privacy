Ringkasan

HoopStat Board adalah aplikasi Android untuk membantu pelatih, asisten pelatih, atau pencatat pertandingan basket dalam mengelola tim, pemain, pertandingan, papan catat digital, statistik, dan laporan pertandingan.

Aplikasi ini dirancang sebagai aplikasi offline-first/local-first. Artinya, data utama aplikasi diproses dan disimpan secara lokal pada perangkat pengguna. Aplikasi tidak membutuhkan login akun, tidak menampilkan iklan, dan tidak mengirim data pertandingan ke server milik developer.

HoopStat Board tidak menjual data pengguna dan tidak menggunakan data pengguna untuk periklanan.

Data yang Diproses di Aplikasi

Untuk menjalankan fitur pencatatan pertandingan, pengguna dapat memasukkan beberapa data secara manual ke dalam aplikasi, seperti:

  • Nama tim, kategori tim, warna tim, dan catatan tim.
  • Nama pemain, nomor jersey, posisi, status starter/bench, dan tim pemain.
  • Data pertandingan, seperti judul pertandingan, tim home/away, mode pertandingan, quarter, durasi, dan status pertandingan.
  • Catatan event pertandingan, seperti 2PT, 3PT, free throw, foul, violation, own basket, waktu game, quarter, pemain, tim, dan koordinat lapangan.
  • Data laporan yang dibuat dari catatan pertandingan, termasuk ringkasan statistik, timeline event, file PDF, dan file CSV.

Data tersebut digunakan untuk menampilkan scoreboard, statistik tim, statistik pemain, timeline event, laporan pertandingan, dan file export yang dipilih oleh pengguna.

Penyimpanan Lokal

Data HoopStat Board disimpan secara lokal di perangkat pengguna menggunakan database lokal aplikasi. Data ini tetap berada di perangkat pengguna selama aplikasi masih terpasang dan data aplikasi tidak dihapus.

Developer tidak memiliki akses langsung ke data lokal pengguna karena aplikasi tidak mengunggah data pertandingan, data tim, data pemain, atau laporan ke server developer.

Jika pengguna menghapus aplikasi atau membersihkan data aplikasi melalui pengaturan perangkat, data lokal seperti tim, pemain, pertandingan, statistik, dan laporan dapat ikut terhapus.

Berbagi PDF, CSV, dan Ringkasan

HoopStat Board menyediakan fitur export dan berbagi laporan pertandingan dalam bentuk PDF, CSV, dan ringkasan teks. File atau teks tersebut hanya dibagikan ketika pengguna secara aktif memilih tombol export atau share di dalam aplikasi.

Saat pengguna membagikan PDF, CSV, atau ringkasan melalui aplikasi lain, seperti WhatsApp, email, Google Drive, atau aplikasi lain yang tersedia di perangkat, proses tersebut mengikuti kebijakan privasi aplikasi pihak ketiga yang dipilih oleh pengguna.

HoopStat Board tidak mengirimkan file laporan secara otomatis ke server developer.

Layanan Pihak Ketiga

Pada versi saat ini, HoopStat Board tidak menggunakan login akun, layanan cloud, iklan, analytics, atau pelacakan pengguna untuk tujuan pemasaran.

Jika pada versi mendatang aplikasi menambahkan layanan pihak ketiga seperti cloud sync, analytics, crash reporting, login, atau iklan, kebijakan privasi ini akan diperbarui agar menjelaskan jenis data yang dikumpulkan, tujuan penggunaan, dan cara data tersebut diproses.

Keamanan Data

Karena data utama aplikasi disimpan secara lokal, keamanan data juga bergantung pada keamanan perangkat pengguna. Pengguna disarankan untuk menggunakan kunci layar, menjaga perangkat dari akses tidak sah, dan tidak membagikan file laporan kepada pihak yang tidak berkepentingan.

HoopStat Board tidak meminta pengguna memasukkan password, akun pribadi, data pembayaran, lokasi GPS, kontak, kamera, mikrofon, atau data sensitif lain yang tidak dibutuhkan untuk fitur pencatatan pertandingan.

Penghapusan Data

Pengguna dapat menghapus data tertentu langsung dari aplikasi, seperti tim, pemain, pertandingan, atau event pertandingan, selama aksi tersebut tersedia dan tidak dikunci oleh aturan riwayat aplikasi.

Pengguna juga dapat menghapus seluruh data lokal aplikasi melalui pengaturan perangkat Android dengan cara membersihkan data aplikasi atau menghapus aplikasi dari perangkat.

Karena HoopStat Board tidak menggunakan akun online dan tidak menyimpan data di server developer, tidak ada data akun online yang perlu diminta untuk dihapus dari server.

Privasi Anak

HoopStat Board ditujukan sebagai alat bantu pencatatan pertandingan dan latihan basket oleh pelatih, asisten pelatih, guru, pembina, atau pengguna yang membutuhkan pencatatan statistik basket.

Aplikasi ini tidak dirancang untuk mengumpulkan data pribadi anak secara online dan tidak menargetkan iklan kepada anak-anak. Jika aplikasi digunakan di lingkungan sekolah atau pelatihan usia muda, pengguna bertanggung jawab untuk memastikan data pemain digunakan secara wajar dan sesuai izin yang diperlukan di lingkungan masing-masing.

Perubahan Kebijakan

Kebijakan privasi ini dapat diperbarui dari waktu ke waktu untuk menyesuaikan perubahan fitur aplikasi, perubahan teknis, atau kebutuhan kebijakan platform distribusi aplikasi.
